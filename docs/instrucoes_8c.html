Descrição Detalhada

Arquivo fonte que monta o vetor de instruções e seus metadados.

A partir de uma enumeração definida no arquivo cabeçalho, este código popula um vetor do tipo "instrucao" atribuindo os valores de nome, opcode, número de argumentos e número de operandos de todas as instruções implementadas neste trabalho.

instrucao* construirInstrucoes ( void  )

Cria um vetor de instruções e as configura.

Após criar um vetor de instruções, a função define cada instrução com os campos corretos (opcode, numarg e numop), retornando o vetor ao fim da configuração.

Observação
Instruções com 'numarg' definido como -1 são instruções com numero de argumentos variáveis, e serão tratadas posteriormente.
Instruções com 'numop' definido como -1 são instruções que precisam verificar o descritor para definir seu número de operandos.
Retorna
Ponteiro para o vetor de instruções já configuradas
