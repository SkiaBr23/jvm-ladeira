Cria um vetor de instruções e as configura. 

Após criar um vetor de instruções, a função define cada instrução com os campos corretos (opcode, numarg e numop), retornando o vetor ao fim da configuração.

\begin{DoxyNote}{Observação}
Instruções com \textquotesingle{}numarg\textquotesingle{} definido como -\/1 são instruções com numero de argumentos variáveis, e serão tratadas posteriormente. 

Instruções com \textquotesingle{}numop\textquotesingle{} definido como -\/1 são instruções que precisam verificar o descritor para definir seu número de operandos. 
\end{DoxyNote}
\begin{DoxyReturn}{Retorna}
Ponteiro para o vetor de instruções já configuradas 
\end{DoxyReturn}
